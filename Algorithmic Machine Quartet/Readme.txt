The Algorithmic Machine Quartet 

What does it do?

Randomly composes 4 part harmony phrase structured compositions.
 
::=> Current Features <=:: 

Randomly choose any of the 12 keys 
Randomly selects the tempo between ( 40..200 )
Randomly choose any one of the 54 scale from scale library

12 pre-built chord progression patterns
